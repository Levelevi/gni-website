Answered
Can you explain the difference between a stack implemented using an array versus a linked list? Which one is better for memory usage?
✅ Prof. Parminder Kaur replied:
Great question, Rajveer! Array-based stacks have fixed size (you must define max size upfront) but offer O(1) access. Linked list stacks are dynamic — they grow as needed — but have extra memory overhead per node for the pointer. For most exam problems, array-based is simpler. For real-world apps where size is unknown, linked list is preferred.

What is the difference between position: absolute and position: fixed in CSS?
✅ Mr. Gurpreet Bhatia replied:
position: absolute places the element relative to its nearest positioned ancestor. position: fixed places it relative to the viewport — it stays in the same spot even when you scroll. Think of a sticky navbar — that uses fixed or sticky. Good question for your exam!
